# Configurando Servidor FTP Seguro em Linux




Instalar Xubuntu 14.04 64bits

Servidor: 107.170.216.161



```bash
apt-get install ssh
apt-get install vsftpd
useradd -m user1 -p user1
useradd -m user2 -p user2


service vsftpd stop
service vsftpd start
```
